## 3. 放置XPシステム

### 3.1 基本仕様

- 放置XPは「現実時間の経過」に応じて自動蓄積される。
- 基本放置XP獲得量: **1分あたり 50 XP**
- オフライン時もカウントされ、復帰時に一括受け取り。
- 放置XPには以下の補正が乗る:
  - 職業ボーナス
    - 映像クリエイター: 放置XP +3%
  - スキル「放置XPアップ」
  - 一時的な時短アイテム／課金ブースト（将来的拡張）

### 3.2 オフラインキャップ

- オフライン蓄積の上限: **24時間分**
  - 24時間を越える分は切り捨て（蓄積しない）。
  - プレイヤーに定期的にアプリを起動してもらうための設計。

### 3.3 放置XPの受け取り

- アプリ起動時：
  - 前回終了時刻と現在時刻の差分を計算。
  - 上限24時間分までをXPとして計算。
  - ポップアップで「放置報酬」として表示（合計XPと、補正の有無など簡易表示）。
- 受け取り演出:
  - XPがゲージに流れ込む簡易アニメーション（0.5〜1秒程度）。

### 3.4 XPとプレイヤーレベル

- プレイヤーレベルはXP総量に応じて上昇。
- レベルアップで：
  - 称号候補の解放（一定Lvごとに上位称号へ）
  - スキルポイント付与（例: Lvアップ毎に1ポイント）
  - 一部システム解放（例: スキルツリーの新ノード、部屋アップグレード開放）

## 4. パズルシステム（1分パズル）

### 4.1 基本仕様

- ジャンル: 落ちもの 3マッチパズル（キャンディクラッシュ型）
- フィールドサイズ: **10列 × 20行**
- 1プレイ時間:
  - 基本: 60秒
  - 職業「プログラマー」: +10秒
  - スキル「パズル時間延長」: +5 / +10 / +20秒
- パズルはあくまで「XPとコインを追加で稼ぐためのミニゲーム」であり、  
  通常の意味でのゲームオーバー（失敗）は存在しない。
  - 制限時間終了 = その時点でのスコアで終了するだけ。

### 4.2 ピース（仕事アイコン）仕様

- ピース種類数: **6種類**
- 生成確率: **完全均等**（各約16.7%）

| ID | 名前              | テーマ/アイコン例            | 備考                          |
|----|-------------------|------------------------------|-------------------------------|
| 1  | ノートPC          | ラップトップ                 | 主人公のメイン作業端末       |
| 2  | ペンタブ          | ペン＋タブレット             | デザイナー感演出              |
| 3  | コーヒーカップ    | カップ＋湯気                 | 色分けしやすい               |
| 4  | USBメモリ         | 棒状ストレージ               | 細長い形でアクセント         |
| 5  | カラーパレット    | パレット＋絵具               | カラフルで視認性が高い       |
| 6  | ビデオカメラ      | 小型カメラ                   | 映像クリエイター要素の表現   |

※ 形は基本的に「正方形 or 角丸」に揃え、色とアイコンで識別しやすくする。

### 4.3 操作・落ち方

- 操作方法: **隣接ピースのスワイプ入れ替え**
  - 上下左右に隣接するピース同士のみ入れ替え可能。
  - 入れ替えで3つ以上揃わない場合は元に戻る（キャンディクラッシュ型）。
- マッチ条件:
  - 縦3以上 or 横3以上連続する同一ピース。
- 消去時挙動:
  - 揃ったピースが同時に消える。
  - 消えたマスの上にあるピースが**一気に下まで落ちる**。
  - 上から新しいピースが補充され、フィールドを完全に埋める。
- 連鎖:
  - 落下＆補充後、新たなマッチがある場合は**自動で連鎖**として処理。
  - プレイヤー操作は不要。
  - 連鎖数に応じて追加スコアボーナス。

### 4.4 スコア計算

- 基本スコア:
  - 3マッチ1回につき **10スコア**
  - 4個以上のマッチは +5〜10程度のボーナス（実装時に微調整可）。
- コンボボーナス:
  - 1連鎖ごとに +5スコア（例: 2連鎖で +5、3連鎖で +10 …）。
- 職業・スキル補正:
  - グラフィックデザイナー職: パズルスコア +10%
  - スキル「パズルスコアUP」:
    - Lv1: +8%
    - Lv2: +15%
    - Lv3: +25%
  - 補正は乗算でまとめて計算（詳細ロジックは実装側で定義）。

### 4.5 パズル結果と報酬

- 終了トリガー: 制限時間終了
- 報酬:
  - XP: `獲得スコア × 10`
    - 例: スコア300 → 3000XP（約1時間放置分）
  - コイン: `獲得スコア × 係数`（例: 0.5〜1.0でバランス調整）
- インタースティシャル広告:
  - パズル3回完了ごとに1回表示（条件: 前回表示から2分以上経過などの保護条件あり）。
- リザルト画面表示項目:
  - 獲得スコア
  - 獲得XP
  - 獲得コイン
  - 連鎖数のベストハイライト
  - （必要に応じて）称号アップ・レベルアップ演出への遷移

## 6. スキルツリー

### 6.1 概要

- 全5種スキル、各スキル3段階強化。
- 形式: **ツリー構造（横方向に枝分かれするシンプルなノード列）**
- スキルポイント:
  - プレイヤーレベルアップ時に1ポイント付与（目安）。
  - スキル解放・強化には「スキルポイント + コイン」が必要。

### 6.2 スキル一覧と効果

#### 6.2.1 放置XPアップ

- 効果: 放置XPに加算の乗算ボーナス。

| Lv | 効果              |
|----|-------------------|
| 1  | 放置XP +10%       |
| 2  | 放置XP +20%       |
| 3  | 放置XP +35%       |

#### 6.2.2 パズルスコアUP

| Lv | 効果                |
|----|---------------------|
| 1  | パズルスコア +8%    |
| 2  | パズルスコア +15%   |
| 3  | パズルスコア +25%   |

#### 6.2.3 パズル時間延長

| Lv | 効果                 |
|----|----------------------|
| 1  | パズル時間 +5秒      |
| 2  | パズル時間 +10秒     |
| 3  | パズル時間 +20秒     |

#### 6.2.4 アクティブスキル「十字消し」

- 効果: 指定マスを中心に上下左右1列を一気に消去する。

| Lv | クールタイム |
|----|--------------|
| 1  | 30秒         |
| 2  | 20秒         |
| 3  | 10秒         |

#### 6.2.5 アクティブスキル「3列横消し」

- 効果: 指定した行と、その上下の行を含む「合計3行」を横方向に全消去。

| Lv | クールタイム |
|----|--------------|
| 1  | 40秒         |
| 2  | 30秒         |
| 3  | 20秒         |

### 6.3 スキルツリー構造（イメージ）

```text
          放置XPアップ
              |
   パズル時間延長  ── パズルスコアUP
              |
      十字消し  ── 3列横消し
```

## 7. 部屋（ワークスペース）システム

### 7.1 概要

- 主人公の「作業部屋」が、レベルに応じて5段階で進化。
- 部屋はステータスには影響しない**見た目のみのスキン**だが、  
  成長の分かりやすい演出として機能する。
- アップグレード対象オブジェクト:
  - PC
  - 机
  - 椅子
  - ポスター
  - 周辺機器（ヘッドホン・マウス・キーボード・ライトなど）

### 7.2 部屋段階のイメージ

1. **Lv1: 雑多なワンルーム**
   - 古いノートPC
   - 小さな木の机
   - 簡易パイプ椅子
   - ポスター無し
   - 背景はやや暗めのブルー

2. **Lv2: 整い始めた作業部屋**
   - ディスプレイ1枚追加
   - メッシュチェアに変更
   - サイドテーブルで机周りが整理される
   - シンプルなポスター1枚

3. **Lv3: クリエイター仕様**
   - ディスプレイ2枚構成
   - ゲーミングマウス・キーボード（光る）
   - ポスター複数枚
   - 背景にライト（RGBライト）追加

4. **Lv4: プロっぽい雰囲気のスタジオ**
   - ウルトラワイドモニター or デュアル大画面
   - 高級チェア（エルゴノミクス）
   - ヘッドホンスタンド ＋ 高品質ヘッドホン
   - デスクライトが強化

5. **Lv5: 最終形態・プロクリエイターのスタジオ**
   - 3画面モニター
   - ハイスペックPCタワー（光る）
   - 壁に賞状や大型ポスター
   - 背景のRGBライトが印象的
   - L字デスクで作業スペース拡大

### 7.3 アップグレード条件・コスト

| 部屋Lv | 必要プレイヤーレベル | 必要コイン |
|--------|----------------------|------------|
| 1 → 2  | Lv3 以上             | 1,000      |
| 2 → 3  | Lv6 以上             | 3,000      |
| 3 → 4  | Lv10 以上            | 8,000      |
| 4 → 5  | Lv15 以上            | 15,000     |

### 8.3 広告仕様

- 表示タイミング:
  - パズル中に「アイテムを追加で使いたい」ときに任意で視聴
- 報酬例:
  - 対象アクティブスキルのクールタイム即リセット
  - 十字消し or 3列消しを1回分追加付与
  - 残り時間 +15秒 など

  - 主人公キャラクター立ち絵
  - 部屋背景（レベルに応じて変化）
  - 部屋タップで「部屋アップグレード」ポップアップ
- 下部:
  - 大きめ中央ボタン: 「パズルを遊ぶ」
  - その左右に:
    - 「スキルツリー」
    - 「ショップ」
- 最下部:
  - バナー広告

#### 10.2.3 パズル画面

- 上部:
  - 左上: 残り時間（数字＋ゲージ）
  - 中央: 現在スコア
  - 右上: ポーズボタン
- 中央:
  - 10×20のパズルフィールド
- 下部:
  - 左下: 十字消しアイコン＋クールタイム表示
  - 右下: 3列消しアイコン＋クールタイム表示
  - 中央下: リワード広告ボタン（有効時のみ表示）

#### 10.2.4 スキルツリー画面

- 上部:
  - タイトル「スキルツリー」
  - 所持スキルポイント表示
- 中央:
  - ツリー状のノード表示（スキルごと）
  - タップでスキル詳細（Lvごとの効果・必要コスト）
- 下部:
  - 「戻る」ボタン

#### 10.2.5 リザルト画面

- 表示項目:
  - 獲得スコア
  - 獲得XP
  - 獲得コイン
  - 連鎖数ハイライト
  - レベルアップ／称号アップがある場合は追従演出
- ボタン:
  - 「部屋に戻る」
  - （将来的に）「もう一度プレイ」

## 11. サウンド仕様

### 11.1 BGM

- タイトル／部屋用BGM:
  - 穏やかでスタイリッシュなループBGMを1曲。
- パズル用BGM:
  - 少しテンポの速いループBGMを1曲。

### 11.2 SE（効果音）

| ID                | タイミング                          |
|-------------------|-------------------------------------|
| se_tap            | ボタンタップ                       |
| se_match          | ピース3マッチ消去                  |
| se_combo          | 連鎖発生時                         |
| se_skill_use      | 十字消し／3列消し使用              |
| se_skill_unlock   | スキル解放時                       |
| se_room_upgrade   | 部屋アップグレード完了             |
| se_level_up       | プレイヤーレベル／称号アップ時     |
| se_result         | パズルのリザルト表示               |
| se_error          | 無効操作（クールタイム中など）     |

## 13. 実装優先度

### Sランク（必須）

- 放置XPシステム（オンライン / オフライン）
- 1分パズルシステム（6ピース・3マッチ）
- 職業・称号システム（3職業 × 程度の称号）
- スキルツリー（5スキル × 3段階）
- 部屋スキン（5段階）
- 基本UI・BGM・SE
- コイン/ジェム/ログボの基本導線
- バナー広告・インタースティシャル広告・リワード広告の組み込み

### Aランク（余裕があれば）

- 細かなパーティクルエフェクト強化
- ローディング演出
- スタートダッシュパック等のスペシャル課金

### Bランク（拡張）

- 職業ごとの専用イベント
- ポートフォリオ画面
- 他プレイヤーの部屋閲覧（オンライン要素）
